PERGUNTAS FREQUÊNTES

Como o GreenBox funciona ? +
O GreenBox é um armário inteligente que simplifica o recebimento de encomendas no condomínio. Quando uma entrega chega, o entregador ou porteiro seleciona o apartamento na tela, deposita a encomenda em um compartimento e o morador recebe uma notificação no WhatsApp com foto da entrega. Para retirar, o morador usa uma senha ou token.
Como é feita a instalação do GreenBox ? +
Nossa equipe técnica realiza a instalação completa, incluindo configuração do sistema e treinamento para porteiros e síndicos. Tudo é feito de forma rápida e sem complicações.
Preciso instalar algum aplicativo ? +
Não! O GreenBox envia notificações diretamente pelo WhatsApp, sem necessidade de baixar aplicativos ou criar cadastros.
O GreenBox funciona sem internet ? +
Sim! O GreenBox possui um servidor próprio que mantém o sistema funcionando mesmo sem internet. Quando a conexão voltar, todas as notificações são enviadas automaticamente.
Como é a segurança do GreenBox ? +
O sistema registra todas as movimentações com fotos, garantindo total transparência. Além disso, apenas moradores autorizados podem retirar encomendas, usando senhas ou tokens.
O GreenBox é de fácil uso para idosos ? +
Sim! Além da senha, oferecemos a opção de tokens físicos, que são práticos e acessíveis para idosos. O sistema também possui áudio que orienta cada passo.
Há fidelidade no contrato ? +
Não! Não exigimos contratos longos ou fidelização. Se o condomínio não quiser mais usar o sistema, basta avisar, e nós removemos o equipamento sem custos adicionais.
E se o entregador não souber usar o GreenBox ? +
O sistema é super intuitivo, e o entregador recebe orientações em áudio e na tela durante o uso. Além disso, os porteiros podem auxiliar no processo.
O que acontece caso o GreenBox fique cheio ? +
O sistema avisa o porteiro quando a capacidade está próxima do limite, permitindo que ele reorganize as encomendas ou notifique os moradores para retirada.
